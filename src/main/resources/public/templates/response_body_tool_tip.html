Inbound parameters can be included in the Response Body using the naming convention:

${REQ_HEAD param_name}
${REQ_PARAM param_name}
${PATH_VAR param_name}

For example, given the URL:   'http://localhost:8001/hello?name=joe'

You could include the request parameter 'name' in your response like so:
{ "msg" : "Hello ${REQ_PARAM name}!" }

...which would return:

{ "msg" : "Hello joe!" }
